# 1337 Backend Assignment
## Features

 - Express [Express](https://expressjs.com/)
 - Logging with [morgan](https://github.com/expressjs/morgan)
 - Axios [Axios](https://www.npmjs.com/package/axios)
 - nodemon [nodemon](https://www.npmjs.com/package/nodemon)
 - jsonwebtoken [jsonwebtoken](https://www.npmjs.com/package/jsonwebtoken)
## Requirements

 - [Node v7.6+](https://nodejs.org/en/download/current/)
 - [Yarn](https://yarnpkg.com/en/docs/install)
 - [NPM](https://www.npmjs.com/)

## Getting Started

#### Clone the repo and make it yours:

```bash
git clone  https://git.heroku.com/be-assignment-1337.git
cd be-assignment-1337
```

#### Install dependencies:

```bash
yarn install Or npm install
```

## Running in local

```bash
yarn start or npm install
```
